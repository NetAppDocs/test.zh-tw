= 改用其他Cloud Central產品
:allow-uri-read: 


[role="lead"]
您可以按一下* Cloud Centr*功能表、然後從列出的NetApp Cloud產品中選取、以切換至其他NetApp雲端產品。您也可以選擇前往Cloud Central入口網站本身。

image:CloudProductsMenu.png["其他雲端產品"]

系統會自動驗證並登入所選的雲端產品。
