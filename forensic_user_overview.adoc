= 鑑識使用者總覽
:allow-uri-read: 


每位使用者的資訊都會在「使用者總覽」中提供。使用這些檢視來瞭解使用者特性、相關實體及最近的活動。



== 使用者設定檔

使用者設定檔資訊包括聯絡資訊和使用者位置。設定檔提供下列資訊：

* 使用者名稱
* 使用者的電子郵件地址
* 使用者管理程式
* 使用者的電話聯絡人
* 使用者位置




== 使用者行為

使用者行為資訊可識別使用者最近執行的活動和作業。這些資訊包括：

* 最近的活動
+
** 上次存取位置
** 活動圖表
** 警示




* 過去七天的營運
+
** 作業數量






== 重新整理時間間隔

使用者清單每12小時重新整理一次。



== 保留政策

如果不再重新整理、使用者清單會保留13個月。13個月後、資料將會刪除。如果Cloud Secure 刪除您的不完整環境、則會刪除與環境相關的所有資料。
