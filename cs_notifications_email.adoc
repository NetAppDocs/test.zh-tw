= 設定警示電子郵件通知
:allow-uri-read: 


[role="lead"]
警示的電子郵件通知會針對警示上的每個動作傳送至警示收件者清單。

若要設定警示收件者、請按一下*管理>通知*、然後輸入每個收件者的電子郵件地址。
