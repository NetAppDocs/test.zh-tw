= 資料收集器清單Cloud Insights
:allow-uri-read: 


[role="lead"]
支援許多廠商與服務的各種資料收集器。Cloud Insights

資料收集器可依下列類型分類：

* 基礎架構：從儲存陣列、交換器、Hypervisor或備份裝置等廠商裝置取得。
* 服務：從Kubernetes或Docker等服務中取得。也稱為_Integrity_。
