:allow-uri-read: 


[role="lead"]
從安裝代理程式的節點收集度量。Cloud Insights



== 安裝

. 從*管理>資料收集器*、選擇作業系統/平台。請注意、安裝任何整合資料收集器（Kubernetes、Docker、Apache等）也會設定節點資料收集。
. 依照指示設定代理程式。這些指示會因您用來收集資料的作業系統或平台類型而有所不同。

== MacOS記憶體使用量

透過Telegraf和MacOS回報不同的記憶體使用量。Cloud InsightsTelegraf和Mac活動監控器都使用從_vm_stat_收集到的數據、不過每個數據的總記憶體使用量的計算方式各不相同。

* Telegraf*計算_使用記憶體總計_如下：

 Memory Used Total = Memory Total - Memory Available Total
其中_可用記憶體總計_衍生自_vm_stat_中的「可用頁數」和「非使用中頁數」總和。

*另一方面、Mac活動監視器*會計算使用的記憶體、如下所示：

 Memory Used = App Memory + Wired Memory + Compressed
其中：

* _App記憶體_衍生自於_VM_stat_中「匿名頁面」與「可清除頁面」之間的差異、
* _Wired Memory_衍生自_VM_stat_中的「連線頁面」、以及
* _Compressed_is衍生自_vm_stat_中的「由壓縮器佔用的頁面」。
